# ドキュメントディレクトリ

このディレクトリには、すべての Markdown ドキュメントファイルが格納されます。

## 構造

- .md ファイルをこのディレクトリに配置してください
- 必要に応じてトピックや部署別にファイルを整理してください
- 新しいファイルを追加する際は、config/config.yaml を更新してください
