このサイトについて

目的

社内手順書・仕様書の一元管理により、情報の散在を防ぎ、効率的な業務遂行をサポートします。

特徴

  • Markdownファイルによる簡単な文書作成
  • タグによる分類・検索機能
  • レスポンシブデザイン
  • GitHub Actionsによる自動デプロイ

アクセス制限

このサイトはプライベートリポジトリとして管理されており、組織メンバーのみがアクセス可能です。
